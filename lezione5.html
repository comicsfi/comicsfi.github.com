Ripassino jQuery

La base di partenza in jQuery sono i selettori, una volta selezionati degli elementi è possibile cambiarne attributi, contenuto, eliminarli, copiarli, modificarli a piacimento

// leggo il nome di un attributo $('selettore').attr('nome_attributo'); // imposto il valore di un attributo $('selettore').attr('nome_attributo','valore_attributo'); // leggo il contenuto $('selettore').html(); // imposto il contenuto $('selettore').html('<p>Posso mettere anche dell\'html qui</p>'); // leggo una proprietà dello stile $('selettore').css('color'); // imposto una proprietà dello stile $('selettore').css('color','orange'); // leggo il valore di un input $('selettore').val();

API

Ogni libreria che vedremo possiede delle API (Application Program Interface), che identificano tutte le funzionalità messe a disposizione dalla libreria stessa e in quale modalità possiamo interagirci. La documentazione per jQuery ad esempio la possiamo vedere su api.jquery.com. Questo è il posto dove cercare le funzionalità di jQuery e gli esempi su come utilizzarle.

Concatenazione di metodi jQuery

jQuery ci offre la possibilità di concatenare metodi, in questo modo il selettore viene ricercato una sola volta cerca una sola volta.



// QUESTO $('selettore').attr('href','http://www.googe.com'); $('selettore').css('color','red'); // puo' essere scritto così $('selettore').attr('href'.'http://www.google.com').css('color','red');

Esercizio


4 elementi. Un input di tipo testo, un div informativo, un bottone cancel e un botton ok.
Premendo OK, devo controllare il valore del campo di testo, nel caso in cui sia vuoto, deve diventare rosso lo sfondo, il bordo e il testo (tonalità differenti altrimenti non si legge), e nel div informativo deve comparire la scritta "Il campo è obbligatorio". Nel caso in cui il campo sia minore di 5 caratteri deve diventare ugualmente rosso, e la scritta nel div deve essere "Il campo deve contenere più di 4 caratteri". Nel caso in cui il campo sia riempito correttamente deve mostrare "Il campo è ok"! Alla pressione di cancel il campo di testo deve essere svuotato.

Toccami

Solitamente gli eventi vengono lanciati con un attributo che identifica l'evento e le sue proprietà. Ad esempio all'evento mousemove viene associato un evento che ha le proprietà clientX e clientY che identificano dove è stato mosso il mouse!


// al movimento del mouse all'interno del body chiamo // la funzione mouse_mosso $('body').on('mousemove',mouse_mosso); // stampo le coordinate del mouse sulla console function mouse_mosso(evento){ console.log("X: " + evento.clientX); console.log("Y: " + evento.clientY); }
