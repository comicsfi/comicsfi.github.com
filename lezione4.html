Esercizio 5

Sapendo che per ottenere la lunghezza di una stringa si utilizza la sua proprietà .length, inizializzare una stringa di nome messaggio assegnandole un valore a piacimento e mostrare il popup "La stringa x lunga n caratteri è lunga!" nel caso in cui la stringa sia più lunga di 4 caratteri e il messaggio "La stringa x lunga n è corta!" nel caso in cui sia più corta, dove x è il valore della stringa mentre n la sua lunghezza in caratteri.
Fate una cosa per volta, non cercate subito la soluzione!

Soluzione 5

messaggio = 'prova'; if( messaggio.length > 4 ) { alert( "La stringa " + messaggio + " lunga " + messaggio.length + " è lunga"); } else { alert( "La stringa " + messaggio + " lunga " + messaggio.length + " è corta"); }
