OGGETTI

Praticamente tutto in Javascript è un oggetto, una stringa, un numero, un array, una funzione sono oggetti.
Inoltre, in javascript potete definire i vostri oggetti.
Un oggetto è un particolare tipo di dato che può contenere proprietà e funzioni (metodi).

// ad esempio per le proprietà var stringa = "ciao"; console.log(stringa.length); var utenti = [ 'admin','user','guest' ]; console.log(studenti.length); // per i metodi // gli oggetti stringa forniscono il metodo toUpperCase stringa.toUpperCase(); // gli array forniscono il metodo sort console.log(utenti.sort());

Creare oggetti

In javascript ci sono tanti modi per creare oggetti!
La sintassi è:

  • Racchiudere tra graffe l'oggetto
  • all'interno delle graffe specificare le proprietà e il loro valore separato da un :

var utente = { nome: "Rocco", cognome: "De Patto", eta: 29 }; console.log(utente.nome); console.log(utente['nome']); console.log(utente.nome.length); var utenti = [ utente, {nome: "Ospite",cognome: "Ospite",eta: 20 } ]; // numero di utenti console.log(utenti.length); // il primo utente console.log(utenti[0].nome); console.log(utenti[0]['nome']); console.log(utenti[0].nome.toUpperCase());

Variabili con funzioni

Le variabili possono contenere oltre che numeri, stringhe, oggetti, array, anche delle funzioni


var prova = function(){ console.log("Sono la funzione prova"); }; prova(); var utente = { nome: "Rocco", cognome: "De Patto", eta: 29, maggiorenne: function(){ return (this.eta>=18); } }; if( utente.maggiorenne() ) console.log("Mostro materiale per adulti"); else console.log("No no no, chiedi il permesso ai genitori");

Creare oggetti

function Utente(nome,cognome,eta) { this.cognome = cognome; this.eta = eta; this.preferenze = preferenze; this.maggiorenne = function(){ return this.eta>=18; }; } var utente1 = new Utente('Rocco','De Patto',29); var utente2 = new Utente('Ospite','Ospite',20); console.log(utente1.nome); console.log(utente2.nome); console.log(utente1.maggiorenne());

Array di oggetti

function Utente(nome,eta,sesso,preferenza){ this.nome=nome; this.eta=eta; this.sesso=sesso; this.preferenza=preferenza; // m f b this.compatibile = function(utente) { if( ( this.preferenza==utente.sesso || this.preferenza=='b' ) && ( utente.preferenza==this.sesso || utente.preferenza=='b' ) ) { return true; } return false; }; } var utente1 = new Utente("Rocco",29,'m','f'); var utente2 = new Utente("Pippo",20,'m','f'); var utente3 = new Utente("Paperina",25,'f','m'); console.log(utente1.compatibile(utente2)); console.log(utente1.compatibile(utente3));

Esercizio

Questo esercizio lo facciamo insieme.

In una pagina ho un form contenente 4 campi, nome, eta, sesso e preferenza sessuale.

Nome è un input testuale, età numerico, sesso una tendina a scelta che puo' essere m o f, preferenza una tendina che puo' valere m, f o b, pensando di default l'utente etero.

Ho due tasti, uno di inserimento, che mi aggiunge un utente (nel caso i campi siano tutti compilati) con le proprietà specificate nel form e uno che una volta premuto mi mostri una lista con tutte le "coppie" possibili!
