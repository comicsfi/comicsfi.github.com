Ricalcheremo quello fatto in php per facilitare la comprensione.
Tenete sempre presente che le nostre App sono sostastanzialmente paragonabili a siti web quindi il nostro tool di riferimento per il test sarà il browser.

Importare script JS da HTML


Come per importare dei fogli di stile in formato CSS, è possibile importare script JS da HTML, dove per il CSS utilizziamo:

<link rel='stylesheet' href='stile.css'>

per importare script JS si utilizza:

<script src='script.js'></script>

Principi di programmazione


Variabili, dichiarazione, tipi.
Contrariamente al PHP non dobbiamo anteporre un $ per identificare le variabili, l'assegnazione è uguale:


a=2; //assegna il valore 2 alla variabile "a" a=2+2; //l'assegnazione può essere anche un'operazione b=a-2; //qui "b" vale 2 (4-2) mentre "a" rimane invariata a='ciao'; //assegno la stringa 'ciao' alla variabile "a"

Operatori numerici

today=1+7; //somma n=1-1; //sottrazione i=1/1; //divisione c=1*1; //moltiplicazione, a=1 today++; //incremento di a, a=2; tomorrow=today++; //restituisce il valore incrementato, a=3, b=3 oggi=++today; //restituisce il valore non incrementato, a=4 b=3

Errori

a=8/0; //Infinity boh++; //boh Undeclared

Usiamo FireBug per trovare gli errori!
