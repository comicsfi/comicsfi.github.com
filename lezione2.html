Ricalcheremo quello fatto in php per facilitare la comprensione.

Importare script JS da HTML


Come per importare dei fogli di stile in formato CSS, è possibile importare script JS da HTML, dove per il CSS utilizziamo:

<link rel='stylesheet' href='stile.css'>

per importare script JS si utilizza:

<script src='script.js'></script>

Principi di programmazione


Variabili, dichiarazione, tipi.
Contrariamente al PHP non dobbiamo anteporre un $ per identificare le variabili, l'assegnazione è uguale:


a=2; //assegna il valore 2 alla variabile "a" a=2+2; //l'assegnazione può essere anche un'operazione b=a-2; //qui "b" vale 2 (4-2) mentre "a" rimane invariata a='ciao'; //assegno la stringa 'ciao' alla variabile "a"

Operatori numerici


today=1+7; //somma n=1-1; //sottrazione i=1/1; //divisione c=1*1; //moltiplicazione, a=1 today++; //incremento di a, a=2; tomorrow=today++; //restituisce il valore incrementato, a=3, b=3 oggi=++today; //restituisce il valore non incrementato, a=4 b=3

Errori


a=8/0; //Infinity boh++; //boh Undeclared

Strutture di controllo, IF e condizioni

Esistono delle strutture di controllo per fare in modo di eseguire solo certe porzioni di codice in base a delle condizioni:

//pseudo-codice if(condizione) fai_questo(); else altro();

Se condizione risulta una verità logica verrà eseguita la funzione di nome "fai_questo()" altrimenti la funzione "altro()";

Strutture di controllo, IF e condizioni

Esempi di condizioni

//assegno un valore alla variabile di nome "utente" nome_utente="Rocco"; //uguaglianza console.log(nome_utente=="Rocco"); //ritorna true console.log(nome_utente=='Marco'); //ritorna false //differenza console.log(nome_utente!='Marco'); //ritorna true console.log(nome_utente!='Rocco'); //ritorna false //esempio utilizzo condizione in struttura if if(nome_utente=='Rocco'){ console.log('Sei l\'insegnante'); } else { console.log('Sei ' + nome_utente); }

Strutture di controllo, IF e condizioni

Esempi di condizioni sui numeri

//assegno un valore alla variabile "n_utenti" n_utenti=3; console.log(n_utenti); //stampo il valore della variabile console.log(n_utenti==3); //controllo l'eguaglianza, ritorna true console.log(n_utenti==2); //controllo l'eguaglianza, ritorna false console.log(n_utenti<5); //uso l'operatore minore (<) ritorna true console.log(n_utenti<=3); //uso l'operatore minore-uguale ritorna true console.log(n_utenti!=3); //ancora l'operatore diverso, ritorna false console.log(n_utenti>2); //ovviamente c'e' anche il maggiore, qui true

IMPORTANTERRIMO!!!
L'operatore di assegnazione non è l'operatore di test di uguaglianza!!
== e = sono costrutti PROFONDAMENTE diversi!
