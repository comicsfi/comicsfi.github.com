Transizioni CSS3

CSS3 introduce nuove funzionalità che ci permettono di animare e produrre effetti prima relegati a flash o a lente funzioni javascript, vediamo di che si tratta.

a { transition-property: all; transition-duration: 1s; }

Supporto dei browser....


a { -webkit-transition-property: all; // chrome / safari -webkit-transition-duration: 1s; -moz-transition-property: all; // firefox -moz-transition-duration: 1s; -o-transition-property: all; // opera -o-transition-duration: 1s; }

Transizioni II

property identifica quale proprietà viene assegnata alla transizione (border, color, background, all, left, all, etc..)

duration definisce la durata...

delay definisce il ritardo per iniziare la transizione

timing-function definisce il tipo di transizione (linear,ease,ease-in...)

è possibile combinare transizioni diverse per gli stessi oggetti....


a { -webkit-transition-property: border, background; -webkit-transition-duration: 1s, 2s; }

Considerate che è possibile inserire racchiudere in classi CSS tutte queste proprietà e assegnare di volta in volta agli elementi del DOM le classi desiderate

Trasformazioni 2D CSS3


Un'altra particolarità introdotta in CSS3 sono le trasformazioni.

a:hover { -webkit-transform: rotate(40deg); -webkit-transform-origin: 0px 0px; }

rotate per ruotare
translate per spostate
scale per scalare l'elemento
skew per distorcere
transform-origin per indicare l'origine della trasformazione

Toccami

Solitamente gli eventi vengono lanciati con un attributo che identifica l'evento e le sue proprietà. Ad esempio all'evento mousemove viene associato un evento che ha le proprietà clientX e clientY che identificano dove è stato mosso il mouse!


// al movimento del mouse all'interno del body chiamo // la funzione mouse_mosso $('body').on('mousemove',mouse_mosso); // stampo le coordinate del mouse sulla console function mouse_mosso(evento){ console.log("X: " + evento.clientX); console.log("Y: " + evento.clientY); }
