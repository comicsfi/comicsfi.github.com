Dati & Strutture dati

Torniamo a parlare di dati.

I dati sono gli oggetti che gestisce, modifica, salva, presenta, crea ogni nostra applicazione.

Il come strutturare i dati è una decisione che spetta a noi in base alla realtà del problema che ci troviamo a modellare di volta in volta.

Oggi vi chiederò di strutturare dei dati in base a varie problematiche, esattamente come si crea prima una tabella con i suoi campi per un database, o come si progetta una biblioteca con i suoi scaffali, i suoi ripiani e i suoi corridoi.
Come potete immaginare questo lavoro viene fatto PRIMA di avere dei dati reali ma è bene tenere in mente degli esempi di dati reali che poi utilizzarenno la nostra struttura, ovvero che poi andranno a riempirla.

Come partire

Gli strumenti che abbiamo a disposizione per strutturare i nostri dati sono le variabili e ne abbiamo solamente di 4 tipi: i numeri (interi o a virgola mobile), le stringhe, gli array e gli oggetti:

I numeri sono autoesplicativi, possono essere sommati, sottratti, divisi, moltiplicati e altro ancora.

Le stringhe sono serie di caratteri in sequenza, contengono parole, frasi, password, nomi, contenuti di email, indirizzi email, quello che scriviamo nella chat di facebook, il titolo di un libro, il vostro cognome.

var n_utenti = 12; function aggiungi_utente(){ n_utenti++; //incremento il numero di utenti } var nome_utente = "Rocco"; //una stringa

Array

Gli array sono liste di altre variabili e possono contenere tutte le altre tipologie di variabili in ogni posizione, possono contenere una lista di nomi di studenti, una lista di cose da fare, una lista di nomi di amici, una lista di titoli di libri, una lista di autori di libri, una lista di editori di libri, una lista di libri. Agli array possiamo aggiungere elementi, togliere elementi, cercare elementi nella lista, clonare una lista....

// creo una lista di STRINGHE! var lista_amici = ['marco','mario','antonio','giovanna','marianna']; console.log(lista_amici[0]); // prende il primo amico lista_amici[0] = 'stefano'; // modifica il primo amico lista_amici.push('pippo'); // aggiunge un amico //restituisce la posizione nella lista dell'elemento cercato console.log(lista_amici.indexOf('pippo'));

Oggetti

Gli oggetti (anche chiamati dizionari) hanno la possibilità di avere un valore per proprietà nominali, possiamo associare ad una proprietà di un oggetto un qualsiasi valore, ad esempio possiamo avere un oggetto utente con le proprietà età (ed un suo valore numerico), nome (una stringa), cognome (una stringa), password (una stringa), amici (una lista di stringhe, o una lista di oggetti utente ognuno con le sue proprietà).

// creo un oggetto chiamato utente con le proprietà nome, cognome, // eta, amici (un array) var utente = { nome: "Rocco", cognome: "De Patto", eta: 29, amici: ['tanti','troppi'] }; // stampo la proprietà nome dell'oggetto utente console.log(utente.nome); // cerco se l'utente di nome Rocco ha un amico chiamato "pippo" // indexOf torna -1 se non trova nulla, altrimenti l'indice del // record trovato nella lista console.log( utente.amici.indexOf('tanti') );

Un passetto avanti

Quando iniziamo a strutturare i nostri dati, dobbiamo sempre tenere presente quali operazioni necessitano i nostri dati, queste operazioni saranno le nostre funzioni.


Abbiamo una lista di utenti? Dobbiamo aggiungerne? Cancellarne? Abilitarli/disabilitarli? Mandargli un messaggio via mail? Dobbiamo capire per ognuna di queste operazioni di quali informazioni necessitiamo per interagire con la nostra struttura dati.

Qualche esempio

Abbiamo una cassettiera con 3 cassetti, questa cassettiera puo' contenere solamente libri. Le operazioni che possiamo effettuare su questa cassettiera sono:

  • Inserire un libro in un cassetto
  • Svuotare un cassetto
  • Mostrare il contenuto di un cassetto
  • Cercare un libro
  • Prendere un libro

La cassettiera

Dunque siccome ogni cassetto puo' contenere una lista di libri, per ogni cassetto avremo un array. quindi il cassetto uno sarà un array di titoli di libri. Possiamo considerare la libreria come un array di cassetti, con all'interno 3 elementi i quali contengono ognuno la loro lista di libri.

/* Inizializzo la cassettiera, con 3 cassetti attualmente vuoti */ var cassettiera = [[],[],[]];

La lista dei libri del primo cassetto è contenuta dentro cassettiera[0], quella del secondo cassetto in cassettiera[1]

Per aggiungere un libro ho bisogno di sapere in che cassetto inserirlo e quale libro sto inserendo, quindi la funzione aggiungi_libro_in_cassetto avrà 2 parametri:


function aggiungi_libro_in_cassetto(titolo_libro, cassetto) { cassettiera[cassetto].push(titolo_libro); }

Provo subito la funzione

// aggiungo un libro nel primo cassetto aggiungi_libro_in_cassetto("Out of control",0); // stampo il contenuto del primo cassetto console.log( cassettiera[0] ); // a questo punto posso fare la funzione che mostra // il contenuto di un cassetto, ho bisogno di sapere solamente // il numero del cassetto function mostra_contenuto_cassetto(cassetto){ console.log(cassettiera[cassetto]); } // e anche quella per svuotare un cassetto function svuota_cassetto(cassetto){ cassettiera[cassetto] = []; }

Cercare..

Cercare un libro in un cassetto è semplice (devo cercare un valore in un array => indexOf)

function cerca_libro_in_cassetto(titolo_libro,cassetto) { var libri_cassetto = cassettiera[cassetto]; var posizione_libro_in_cassetto = libri_cassetto.indexOf(titolo_libro); if(posizione_libro_in_cassetto>-1) { console.log("Libro trovato alla posizione " + posizione_libro_in_cassetto + " nel cassetto " + cassetto); } else { console.log("Libro non trovato nel cassetto " + cassetto); } }
