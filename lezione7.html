jQuery / Metodi

Una volta selezionati degli elementi è possibile utilizzare i metodi jQuery per interrogarne proprietà e contenuti e ovviamente modificarli, cancellarli, aggiungerne..

//esempi dei metodi più comuni, ricordate sempre http://api.jquery.com $("selettore").html(); //legge il contenuto html $("selettore").html("contenuto"); //imposta il contenuto $("selettore").attr('href'); //legge un attributo $("selettore").attr('href','http://google.com'); //imposta un attributo $("selettore").css('color'); //legge una proprietà css $('selettore').css('color','#FFF'); //imposta una proprietà css // legge o scrive il valore degli input/select/textarea $('selettore').val();

Cercate e provate i metodi hasClass, addClass, removeClass, toggleClass. Esercizio, implementiamo la funzione toggleClass come se non esistesse già, solamente attraverso addClass, removeClass, hasClass

jQuery / Eventi

L'esecuzione del nostro codice è orienta agli eventi, eseguiamo operazioni al succedere di particolari condizioni, alla pressione di un tasto, quando l'utente sposta il mouse in un particolare posto, al download completato...

Per legare l'esecuzione di una funzione ad un evento usiamo il metodo .on di jQuery:

// in questo modo diciamo di eseguire la funzione nome_funzione // quando verrà scatenato l'evento nome_evento sugli elementi // selezionati dal selettore specificato $('selettore').on('nome_evento',nome_funzione);

Quasi tutti gli eventi hanno una funzione specifica per cui possono essere utilizzati senza il .on(), ma con il nome dell'evento stesso:

//esegui nome_funzione al click di ogni elemento selezionato $('selettore').click(nome_funzione);
