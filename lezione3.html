Interagire con il DOM


  • La prima funzionalità importante di javascript che impareremo sarà quella di selezionare elementi nel DOM.
  • Selezionando un elemento sarà possibile cambiarne tutte le proprietà, il contenuto, lo stile, etc..
  • Useremo dal principio i metodi nativi del javascript per poi utilizzare una libreria come JQuery

Oggetto document


  • L'oggetto document è un oggetto presente in ogni documento HTML
  • Possiede moltissime proprietà tante delle quali modificabili
console.log(document.title); console.log(document.domain); console.log(document.width); console.log(document.height);
  • ...e possiede delle funzioni..
//seleziono un elemento nel DOM document.getElementById('id_elemento');

Elementi nativi

HTML

...doctype / head ... <body> <div id='id_elemento'> </div> </body>

JS

//seleziono un elemento var elemento = document.getElementById('id_elemento'); console.log(elemento); //cambio il colore dell'elemento selezionato elemento.style.color = 'orange'; //cambio il contenuto dell'elemento selezionato elemento.innerHTML = 'Prova contenuto dinamico';

JQuery


  • JQuery è bello, JQuery è meglio
  • Non dobbiamo patire le differenze dei vari browser
  • USIAMO I SELETTORI CSS PER SELEZIONARE GLI ELEMENTI!!
$('#id_elemento').css('color','orange'); $('#id_elemento').html('Prova contenuto dinamico');

JS non è multithreading


  • Possiamo fare solamente una cosa alla volta.
  • Quindi, il nostro codice NON deve essere bloccante, altrimenti si ferma il browser!!
  • Esistono delle eccezioni, possiamo gestire delle operazioni asincrone come il download...
  • ... o attivare un timer
//l'oggetto window e un esempio di proprietà console.log(window.screenLeft); //una funzione di prova function prova(){ console.log("Dentro la funzione prova"); } setTimeout(prova,2000);

setInterval


//solita funzione di prova function prova(){ console.log('Dentro la funzione di prova'); } setInterval(prova,1000);

Ultimo esempio e poi cominciamo sul serio

// memorizzo da qualche parte a che step sono arrivato var step_n = 0; var max_step = 10; var timer; //solita funzione di prova function prova(){ console.log("Sono allo step " + step_n++); if(step_n==max_step) clearInterval(timer); } timer = setInterval(prova,500);
